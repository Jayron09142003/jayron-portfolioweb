"SafeStep" Capstone Project

Mobile Development and IoT Application

Our capstone project, titled GPS-Enabled Obstacle Detection and Navigation System with Multi-Sensory Feedback for Visually Impaired, is a mobile-based assistive technology solution designed to help the visually impaired navigate their surroundings safely and independently. Unlike traditional systems that rely solely on one type of feedback, our solution integrates multiple technologies to enhance usability and effectiveness. It uses the smartphone‚Äôs built-in camera for real-time object detection, GPS for location awareness, and provides multi-sensory feedback through vibrations and voice alerts. Additionally, we integrated IoT devices such as ultrasonic sensors and the ESP32-CAM module to support basic image processing and obstacle detection in real-time. This hybrid approach allows the system to detect nearby objects and deliver intuitive alerts, enabling users to navigate both familiar and unfamiliar environments with greater confidence. The project aims to improve accessibility, reduce dependency, and promote inclusivity through practical, affordable, and scalable technology.
