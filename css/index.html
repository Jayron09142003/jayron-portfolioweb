IT Projects and Capstone

"Spate" an E-Commerce Platform

A Full Stack Development

Spate is a minimalist eCommerce website built using ASP.NET for its backend framework and Microsoft Access as its database. Designed with simplicity and user experience in mind, Spate offers a clean, distraction-free interface that focuses on functionality and ease of navigation. The platform includes core eCommerce features such as product listing, cart management, and a secure checkout system. It leverages ASP.NET’s robust server-side capabilities for handling dynamic content and form submissions, while MS Access manages product data, user records, and order transactions. Spate was developed to demonstrate how lightweight technologies can be used to build fully functional web applications. Its minimalistic design ensures fast load times and an intuitive browsing experience across devices, making it ideal for small businesses or startups looking for a streamlined digital storefront.

"SafeStep" Capstone Project

Mobile Development and IoT Application

Our capstone project, titled GPS-Enabled Obstacle Detection and Navigation System with Multi-Sensory Feedback for Visually Impaired, is a mobile-based assistive technology solution designed to help the visually impaired navigate their surroundings safely and independently. Unlike traditional systems that rely solely on one type of feedback, our solution integrates multiple technologies to enhance usability and effectiveness. It uses the smartphone’s built-in camera for real-time object detection, GPS for location awareness, and provides multi-sensory feedback through vibrations and voice alerts. Additionally, we integrated IoT devices such as ultrasonic sensors and the ESP32-CAM module to support basic image processing and obstacle detection in real-time. This hybrid approach allows the system to detect nearby objects and deliver intuitive alerts, enabling users to navigate both familiar and unfamiliar environments with greater confidence. The project aims to improve accessibility, reduce dependency, and promote inclusivity through practical, affordable, and scalable technology.

"Street Sprinter"

Game Development

Street Sprinter is a browser-based endless runner game inspired by Google Chrome's offline dinosaur game, enhanced with modern features and a pixel-art aesthetic. Built using Web Forms and basic file handling with Notepad as the data storage, the game includes multiple characters, increasing difficulty levels, and a leaderboard system to track high scores. Players can enjoy a vibrant, retro-style experience with colorful pop culture-inspired sprites, responsive controls, and dynamic obstacles. The game also features score tracking, limited lives, and visually engaging pixel animations, blending nostalgic charm with upgraded functionality.

"Cinema Snack Menu"

Stack Application

As part of our Data Structures course, Me and my Groupmates developed a console-based application in C++ called the Cinema Snack Menu to demonstrate the use of the stack data structure. The project simulates a simple snack ordering system where users can add items to their order, view their current stack of snacks, and undo the most recent selection. It applies the Last-In, First-Out (LIFO) principle of stacks, showing how fundamental data structures can be used to solve real-world problems in an intuitive and engaging way.
